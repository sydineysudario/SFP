🚀 Configuração do Firebase

  1. Acesse Firebase Console
  2. Crie um novo projeto ou use um existente
  3. Ative Authentication > Sign-in method > Google
  4. Ative Firestore Database
  5. Em Project Settings > General, copie suas credenciais
  6. Substitua o conteúdo do script JSON com suas credenciais
  7. Em Authentication > Settings > Authorized domains, adicione seus domínios

📊 Estrutura do Firestore

users/{uid}/
├── transacoes/
│   ├── {docId}
│   │   ├── tipo: "receita" | "despesa"
│   │   ├── descricao: string
│   │   ├── valor: number
│   │   ├── categoria: string
│   │   ├── data: string (YYYY-MM-DD)
│   │   └── createdAt: timestamp
│   └── ...
└── patrimonio/
    ├── {docId}
    │   ├── tipo: "bem" | "divida"
    │   ├── descricao: string
    │   ├── valor: number
    │   ├── categoria: string
    │   └── createdAt: timestamp
    └── ...

🌐 Deploy

GitHub Pages:

  1. Faça upload do arquivo HTML para um repositório
  2. Ative GitHub Pages
  3. Adicione o domínio do GitHub Pages aos domínios autorizados do Firebase

Vercel:

  1. Conecte seu repositório ao Vercel
  2. Deploy automático
  3. Adicione o domínio do Vercel aos domínios autorizados

Firebase Hosting:

  1. npm install -g firebase-tools
  2. firebase login
  3. firebase init hosting
  4. firebase deploy
